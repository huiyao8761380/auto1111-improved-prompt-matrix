## Usage

Use `<` `>` to create a group of alternate texts. Separate text options with `|`. For example:

An input of`,<short|long|messy> hair,` 

Will output random prompts: `,short hair,`,`,long hair,`,`,messy hair,`

`Batch count`>=1,`Batch size`=1
